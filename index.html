Understand margin, padding & borders

This page demonstrates how the CSS Box Model affects spacing and sizing. Resize the window to see layout changes.

Padding

Padding increases the space inside an element, between its content and its border.

Margin

Margin controls space outside the element and separates it from other elements.

Border

Border is the visible line around an element that adds thickness and style.
